MMU Online Forum #HTML, ​#Bootstrap, ​#CSS, ​#JavaScript, ​#PHP

This forum is created for Multimedia University Cyberjaya

  • to create a proper platform that provides information to the public.
  • to provide an organized internet forum.
  • to provide an online classified ads page for MMU students.

This forum works based on different access levels of users, which are

PUBLIC VISITOR MMU-ians ADMIN BLOCKED 


As a PUBLIC user_

As you can see, users can view the forum without login or sign up. Public users can still view, sort and filter topic list, but they cannot create a new topic. Same for the comments, public users can view others comments, but they cannot create a new comment.


To become a VISITOR user_

To become a VISITOR, just need to fill in the required field to sign up.


As a VISITOR user_

As a VISITOR, users can comment and create topic now. They can report others' topic if they found out that it is inappropriate. They are able to delete their own topic too.
However, a VISITOR has no access to "SHOP".


To become a MMU-ians user_

Every user who signed up has their own profile with their basic information and topics they posted. To change the status from VISITOR to MMU-ians, users need to upload their MMU's ID photo for admin to verify.

The admin will then approve the request if it is an MMU's ID.


As a MMU-ians user_

After the status changed to MMU-ians, users will realise they actually have access to a new division which is "SHOP". They can view others' advertisement in the "SHOP" division now. Also, they can submit the ads they want to post in "SHOP" for the admin to approve.


As an ADMIN _

There is a drop-down navigation for ADMIN, inside, there are

  • ID VERIFICATION
     Approve a VISITOR to become MMU-ians if the photo uploaded is MMU's ID card.

  • SHOP APPROVAL
     Review ads submitted by MMU-ians before it posted in "SHOP" division to prevent inappropriate ads.

  • REPORT
     Review users' reports and decide what action needed to be done to the topic or user.

  • BLOCKED USER
     An ADMIN can block a user, and of course, an ADMIN has a list to review and unblock them.

BLOCKED _

However, don't let others report you too frequent, else you might meet this situation.
